# 一个简洁风的中文学术简历模板

这是一个简单易用的简洁风学术简历模板，适用于研究生、博士的申请。整个排版简洁干净，没有五颜六色的颜色干扰。LaTeX内置的超链接能够让导师一键跳转你的个人主页，一键发送邮件，在一大堆简历中脱颖而出！

模板如下，整体使用xeLatex编译，每个内容都可修改，图标使用的是fontawesome5宏包，内置超多图标，保证有符合你心意的一个！
